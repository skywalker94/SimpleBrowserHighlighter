Setup Guide

  1. From the releases page of the GitHub repository, 'Downloads' linked above, download the ZIP (.zip) file of the latest release.
  2. Extract the files into your folder of choice. This will become the location where you host the chrome extension.
  3. Navigate to chrome://extensions/ in the browser. You can also find this by clicking the menu button and looking for Manage Extensions.
  4. Enable Developer mode, if you have not done this before. It is usually found in the top right corner.
  5. Click Load unpacked and select the folder you extracted the ZIP files into in step two. The chosen folder should contain manifest.json.

Updates

Fetch a the new version from the 'Downloads' above. Replace the existing files in the folder that hosts your extension. Click the reload icon on the extension card in chrome://extensions/.

FAQ

Is my data sent to a server?

All highlights and color preferences are stored locally in your browser's storage. Nothing is tracked.

How do I change the shortcut?

Open chrome://extensions/shortcuts and set the shortcut for "Simple Highlighter" if Ctrl+Shift+H is taken.

Will changes to the base page affect my highlights?

It is possible. Since it relies on the page's structure and content, any significant changes may impact highlight behavior.
